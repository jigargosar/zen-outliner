Workflowy Keyboard Shortcuts Reference

Complete reference of Workflowy's keyboard shortcuts, organized by category.
Used as the feature map for zen-outliner — each shortcut implies a feature.

Win = Windows/Linux keybinding, Mac = macOS keybinding.


# Core Editing

+-----+==========================+===================+===================+============================+
| #   | Action                   | Win               | Mac               | Notes                      |
+-----+==========================+===================+===================+============================+
| 1   | New sibling below        | Enter             | Enter             | Creates empty node after   |
|     |                          |                   |                   | current                    |
+-----+--------------------------+-------------------+-------------------+----------------------------+
| 2   | Split node at cursor     | Enter (mid-text)  | Enter (mid-text)  | Text after cursor becomes  |
|     |                          |                   |                   | new sibling below          |
+-----+--------------------------+-------------------+-------------------+----------------------------+
| 3   | Delete empty node        | Backspace (empty) | Backspace (empty) | Removes node, focuses      |
|     |                          |                   |                   | previous sibling           |
+-----+--------------------------+-------------------+-------------------+----------------------------+
| 4   | Merge with previous      | Backspace (start  | Backspace (start  | Appends current text to    |
|     |                          | of non-empty)     | of non-empty)     | previous sibling's end     |
+-----+--------------------------+-------------------+-------------------+----------------------------+
| 5   | Delete node + children   | Ctrl+Shift+Bksp   | Cmd+Shift+Bksp    | Force-deletes regardless   |
|     |                          |                   |                   | of content                 |
+-----+--------------------------+-------------------+-------------------+----------------------------+
| 6   | Add a note               | Shift+Enter       | Shift+Enter       | Opens note area below      |
|     |                          |                   |                   | node text                  |
+-----+--------------------------+-------------------+-------------------+----------------------------+
| 7   | Mark complete            | Ctrl+Enter        | Cmd+Enter         | Strikethrough toggle       |
+-----+--------------------------+-------------------+-------------------+----------------------------+
| 8   | Duplicate node           | Ctrl+Shift+D      | Cmd+Shift+D       | Copies node below with     |
|     |                          |                   |                   | #copy tag                  |
+-----+--------------------------+-------------------+-------------------+----------------------------+
